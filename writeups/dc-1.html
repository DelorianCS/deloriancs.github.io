DC-1

Status: Completed

Enumeration

Started with host discovery to identify the target's IP address on the network. A full service/version scan revealed open ports: SSH (22), HTTP (80), and RPC (111). Browsing to port 80 presented a web server running Drupal.

Exploitation

Used the Drupalgeddon2 exploit to gain remote code execution. After configuring the exploit in Metasploit, a successful meterpreter session was obtained. Within the shell, retrieved the initial flag and discovered further Drupal configuration in sites/default/settings.php.

Accessed the MySQL database using known credentials, enumerated the Drupal tables, and extracted password hashes. After cracking them, used the credentials to log in to the dashboard and obtain another flag.

Privilege Escalation

Back in the meterpreter shell, explored the system manually. Discovered a user home directory and searched for binaries with SUID permissions. Using one of these, escalated to root. From there, accessed the root directory and retrieved the final flag.
