Enumeration

I started with an aggressive nmap scan to identify open ports and running services

Found port 80 open so I visited the web page

I tested default credentials and received an invalid login message

I attempted SQL injection using Burp Suite to intercept and modify the request

Used Burp Intruder and tested payloads like ' OR '2'='2'-- which produced a server error

I reset the machine and moved to a new approach

Used SecLists to brute-force usernames

Discovered a valid user: martin

Tested payload martin' AND ''=''-- - to bypass authentication

Successfully logged in

Exploiting

The injection was possible due to insecure SQL parsing in the login form

Used boolean-based payload instead of classic OR 1=1 for safer exploitation

Confirmed login and accessed the user interface
