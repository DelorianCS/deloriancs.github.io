Blue

Status: Completed

Enumeration

Started by scanning the target with nmap, using version detection and vulnerability script category to check if its vulnerable to ms17-010.

The scan revealed multiple open ports under 1000. Among the findings, port 445 (SMB) indicated the target was vulnerable to ms17-010 (EternalBlue).

Exploitation

Used Metasploit to search and load the exploit module for ms17-010. Configured the required target settings including RHOST and payload for reverse shell.

Launched the exploit and successfully received a shell back.

Privilege Escalation

Once inside, verified access and explored user directories for flags. Since the exploit provided NT AUTHORITY\SYSTEM privileges, no further escalation was required.

Retrieved user.txt from the user profile directory and root.txt from the Administrator's desktop.
