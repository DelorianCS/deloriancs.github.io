Weak RSA

Status: Completed

Enumeration

Used nmap to scan the target. Identified FTP on port 21 and Apache HTTP server on port 80. Default scripts helped detect versions like vsftpd 3.0.3 and apache httpd 2.4.41.

FTP service allowed anonymous login. Used standard client to connect and explore the available files. Downloaded allowed.userlist and allowed.userlist.passwd from the server.

Exploitation

Ran gobuster to enumerate web directories. Discovered login.php which exposed a login interface. Tested username and password combinations from the FTP dump.

Successfully logged in using one of the privileged users listed in the FTP files. Gained access to the internal admin page which contained the final target information.

Privilege Escalation

This challenge focused solely on web exploitation and credential discovery. No shell access or further escalation was needed. After logging in with valid credentials, access was granted to the sensitive information.
