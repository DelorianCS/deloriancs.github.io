Brooklyn Nine-Nine

Status: Completed

Enumeration

Started with a VPN connection and basic checks to ensure the target was live. Used nmap to scan for open ports. Found FTP and SSH available, along with an accessible HTTP service.

Connected to the FTP service and downloaded a file named note_to_jake.txt, which hinted at a weak password for user jake.

Exploitation

Used hydra to brute-force SSH credentials for user jake. Successfully found a working password. Logged in via SSH to gain initial access.

Explored Jake's home directory and retrieved the first flag, confirming initial user-level access.

Privilege Escalation

Investigated the system and identified potential escalation paths. One approach involved steganography techniques, possibly tied to hidden files or images found on the system.

Eventually, root access was achieved after deeper enumeration and exploitation of overlooked files. Retrieved the root flag to complete the room.
